# IR_send_read
Emisor/Receptor IR Aires -TV 

Dependencias:

  - Adafruit_GFX.h
  - Adafruit_SSD1306.h
  - IRremote.h / hpp  (Armin Joachimsmeyer)
  
Hardware:

  - Arduino NANO (Old bootloader).
  - OLED I2C 128x32 
  - IR reader de tres terminales.
  - IR emisor.
  - Tres botones pulsadores.
  - Diodo led rojo.
